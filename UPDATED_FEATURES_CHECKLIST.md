### File Organization & Display

- [ ] Sort by last used date (My Files)

- [ ] Multiple formats: PDF, Excel, Word, JPG, all printable formats

- [ ] Maker creates ‚Üí Checker reviews
- [ ] Checker sees 3 buttons: Check, Approve, Revert

- [ ] MD approval with offline upload
- [ ] Any maker/checker can upload physical scanned approval
- [ ] Close note after uploading offline approval
- [ ] Digital signature support
- [ ] Print option after final approval
- [ ] Print includes: approval sign, date, time, approved by, location
- [ ] All approvals shown: maker, checker, final approver

- [ ] Actionable department receives file after approval
- [ ] They add implementation comments

- [ ] Report section with file logs

- [ ] DMS integration for fetching old files
